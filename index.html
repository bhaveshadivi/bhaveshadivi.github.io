My Projects

MelanoScan: A Cutting Edge Diagnostic Tool for Early Melanoma Detection

An Python ML + Enzyme Kinetics-based Experimental Dual-System (Arduino & C++) that is meant to analyze the suspected lesion microenvironment for KLK14 Protease and CDSN metabolic ratios and alignment with key superficial features as per the visual ABCDE framework for detecting metastatic Melanoma

DistilMind: a Mental Health NLP Model for Enhanced Sentiment Analysis (Tensorflow)

QLoRA-fine-tuned BERT model analyzing sentiment + wellness indicators for potential mental health risks (purely experimental)

Pangolin: LLMs + HTTP requests + UI/UX

An enhanced User-Interface for calling Large-Language-Models using the Streamlit Framework + ngrok in IPython

Ionite: a cron + Vercel-based workflow to check for school club spots

Built in backend Python, Flask endpoint hosted by Vercel, and pinged by cron. Checks periodically if a club spot is available for a specific date through my school's intranet (ION v3) API
